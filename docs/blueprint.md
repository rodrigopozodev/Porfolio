# **App Name**: PortfolioPulse

## Core Features:

- Hero Section: Display personal information, including name, profession, and a brief introduction.
- Projects Section: Showcase recent projects with titles, descriptions, technologies used, and links to code and live demos.
- Skills Section: List key skills with icons or logos for each skill.

## Style Guidelines:

- Primary color: Use a clean white or light gray for the background.
- Secondary color: A darker gray or muted blue for text and subtle elements.
- Accent: Teal (#008080) for interactive elements, buttons, and highlights.
- Use a responsive grid layout to adapt to different screen sizes.
- Use a consistent and modern icon set (e.g., Lucide Icons) for visual appeal.
- Subtle transitions and animations for a polished user experience.
